# Velkynvelve
> **Type:** Drow Outpost / Prison
> **Region:** The Northdark
> **Significance:** Campaign starting location

## Description
Drow military outpost where the party was held prisoner at the start of the campaign. The party escaped during S03 and returned voluntarily in S11.

## Campaign Events
- **S01–S02:** Party held as prisoners (~8–9 days captivity)
- **S03:** Escape from Velkynvelve
- **S11–S13:** Party returns armed, linked via spore communication
