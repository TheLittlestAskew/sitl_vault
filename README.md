A living archive for our D&D 5E (2024) campaign set in the Underdark. Everything here is generated from our session recordings, transcripts, and D&D Beyond data — organized in an Obsidian vault and synced to GitHub so the whole group can browse it.

## How This Gets Made

After each session, the workflow goes roughly like this:

1. **Recording** gets transcribed using AssemblyAI with campaign-specific vocabulary boosting (so it actually recognizes "Velkynvelve" instead of "Belkinvelve")
2. **Transcript** gets spell-checked against D&D source material and formatted
3. **D&D Beyond roll data** gets synced from the game log into a database
4. **Session notes** are generated — cross-referencing transcript, roll archive, and prior sessions
5. **Vault files** get updated — session page, character pages, NPC pages, locations, loot, quotes, the dashboard, everything

The whole thing is AI-assisted (Claude) but human-verified. Taylor reviews and approves every step. Nothing gets published without a check.

## How To Browse

You can read everything right here on GitHub by clicking into the folders above. The `.md` files render nicely in GitHub's viewer.

If you want the full experience with backlinks and graph view, you can clone the repo and open it in [Obsidian](https://obsidian.md/) (free). The internal `[[backlinks]]` connect characters to sessions to locations — it's designed to be explored that way.

## What's NOT Here

A few things are kept local and not pushed to GitHub:
- **Session recordings** — audio files with real people's voices stay private
- **Copyrighted D&D PDFs** — Player's Handbook, etc. Buy your own books, nerds.
- **API keys and config files** — boring infrastructure stuff

## A Note on Accuracy

This archive follows a strict set of rules:
- **No invention.** Every event, quote, and detail comes directly from the transcript or roll log. If something is ambiguous or unclear, it gets flagged rather than guessed at.
- **Verbatim quotes only.** Dialogue is never paraphrased.
- **DM authority is final.** Addison's rulings override published rules. Always.
- **Metagaming is excluded.** Kit's journal only contains things Kit could know, see, or feel from inside the story.

If you spot an error, tell Taylor.
